标签模板

2019-10-17
JS 新知识
约 395 字 预计阅读 1 分钟

标签模板其实不是模板，而是函数调用的一种特殊形式。“标签”指的是函数，紧跟在后面的模板字符串就是它的参数。

 var a =5
 var b=10

 xx `Hello ${a+b} world ${a*b}` //xx是函数名字，这句话是函数调用

 function xx(){
     console.log(arguments[0],arguments[1],arguments[2],arguments[3])
 } //打印出[ 'Hello ', ' world ', '' ] 15 50 undefined  //只传了两个模板字符串进来，所以第四个为undefined

xx函数的第一个参数是一个数组，该数组的成员时模板字符串中那些没有变量替换的部分，也就是说，变量替换只发生在数组的第一个成员和第二个成员之间，以此类推。xx函数的其他参数都是模板字符串各个变量被替换后的值，由于本例中，模板字符串含有两个变量，因此tag会接收到value1和value2两个参数。

xx函数所有参数的实际值如下：

——第一个参数：[‘Hello ‘,’ world ‘,“]

——第二个参数：15

——第三个参数：50
