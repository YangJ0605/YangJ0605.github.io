es6之set

2019-10-18
JS 新知识
约 359 字 预计阅读 1 分钟

set是ES6中新增的类型，和数组类似，唯一不同在于该类型不会有重复的数据，一般常用来对数据进行去重操作。

一、声明

let set = new Set();即创建了一个空的set

二、赋值

1
2
3
4
5
6
7

let set = new Set([1,2,3,4])
console.log(set) //输出一个对象，属性0存着1，属性1存着2，....,length为4 （一个伪数组）

let set = new Set([1,2,3,4,2,2,3,4,1,5])
console.log(set)  //输出Set { 1, 2, 3, 4, 5 }  ,自动去重

三、属性

size属性求set集合的大小(长度)

1
2

let set = new Set([1,2,3,4])
set.size===4

四、方法

1.add方法，表示新增一个元素到set中。

set.add('100') 则输出Set { 1, 2, 3, 4, '100' },注意是字符串！

2.delete方法，表示删除一个元素，返回的结果为true或者false。

3.has方法，表示判断某个元素是否在set这个集合中，返回值为true，false。

4.clear方法表示清除集合中所有的元素。没有任何返回值。直接清除元素
