常见的一些web性能优化套路

1.减少DNS查询

2.TCP连接复用（keep alive）、HTTP2.0 多复用

3.发起请求的时候减少cookie，使用 Cache-Control

4.接受响应时 使用 Etag 使用 Gzip

5.合并文件

6.调整 CSS 和 JS 的位置

7.压缩图片，使用雪碧图

8.增加域名以并行下载资源
